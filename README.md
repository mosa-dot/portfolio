## Project 1: MarqFit (Flutter + Firebase)
**What it is:** Cross-platform fitness app for Marquette community.  
**Users / problem:** Students need quick visibility into gym occupancy + simple logging.  
**My role:** Full-stack developer & founder (end-to-end).  
**What I built:**
- Flutter mobile UI + state management
- Firebase Auth + Firestore backend (real-time updates)
- Core features: occupancy, logging, social components (beta)
**UX decisions & iteration:**
- Iterated navigation/workflows from feedback from 20+ beta users
- Fixed layout/render issues to improve consistency across devices
